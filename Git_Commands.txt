
# Git Commands
1. git add .
2. git commit -m "first"
3. git push origin main
4. git commit -ac "statement to be commited" - used to perform add and commit at a time.

5. git status - used to check current status of chamges. 

6. git branch - used to check current branch..

7. git checkout "branch name" - used to switch branch..

8. git checkout -b 'branch name' - used to create new branch..

9. git branch -d 'branch name' - used to  delete a branch..

10. git push --set-upstream origin 'branch name' - used to set new upstream branch

11. git merge 'branch name' - used to merge upstream branch with main branch

12. git log - used to see all changes with commit.

13. git reset 'filename' - used to reverse most modified change in all files.

14. git reset HEAD~1 - used to reverse the staged change.

15. git reset --hard hashID - used to make permanent changes in code till committed Id.

16. git diff 'branch name' - used to compare two branches
